Сменить IP адрес у абонента

Смена IP выполняются только в исключительных формах<br />

<ul><li>Открываем карточку абонента в <b>RS</b></li><li>Необходимо отвязать порт на коммутаторе (<font color="#f44336">не забыть какой коммутатор и порт</font>) вкладку можно оставить</li><li>Проходим в раздел <b>IP-адрес </b></li><li>Дублируем страницу в новой вкладке </li><li>В первой странице заходим изменение IP-адрес</li><li>Основные данные <b>убираем галочку</b> (IP адрес станет неактивным) </li><li>Вырезаем <b>MAC-адрес</b></li><li>Вырезам <b>Сетевое имя</b></li><li>В дублированной вкладке нажимаем <b>Добавить IP адрес</b> </li><li>Вставляем в дублированную вкладку <b>Вырезанный MAC</b> адрес и <b>Сетевое время</b></li><li>Группа* указываем нужный нами микрорайон. </li><li>Добавляем свободный <b>IP адрес</b></li><li>Сохраняем изменения в <u>первой вкладке</u> и только потом сохраняем в <u>дублированной вкладке</u></li><li>Привязываем учетную запись к прежнему порту. </li></ul><div>Если к примеру при заключении договора был неверно указан адрес, к примеру другой микрорайон, то после всех данных манипуляции удаляем старый IP и меняем в карточке абонента группу микрорайонов. <br /></div>

Как найти свободный IP адрес<br />

<ul><li>Открываем RS</li><li>Справочники</li><li>IP-адреса </li><li>В фильтре выбираем нужный микрорайон</li><li>Ищем последний зарегистрированный IP адрес нового абонента и добавляем +1, проверяем свободен ли IP адрес введя его в RS</li></ul><div><br /></div><div>Также можно использовать резервные IP адреса. 10.*.*.10, 10.*.*.254, 10.*.*.255</div><div> <br /></div>
