<p>Специалист технической поддержки должен:</p>
						<ul>
							<li>Отвечать на обращения из автоответчика, перезванивая на номер оставленный в обращении на  Dashboard RS<br /></li>
							<li>Отвечать на сообщения абонентов из социальных сетей поступающие в битрикс через модуль Открытые линии</li>
							<li>Консультировать по телефону абонентов</li>
							<li>Создавать наряд на проверку/ремонт нашего оборудования или кабеля, после выяснения проблемы по телефону</li>
							<li>Отвечать на письма абонентов на почте через Битрикс</li>
							<li>Настраивать роутеры абонентов</li><li>Обязан уточнять и обновлять контактные данные абонентов при каждом разговоре! Актуализация контактных данных</li>
						</ul>
