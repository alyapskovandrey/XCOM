# Конфигурирование ПК технического специалиста

Список программ которые должны быть установлены на рабочем ПК технического специалиста <br />

1) Операционная система - Windows 10 с отключенной функцией авто-обновления<br />2) Браузер для работы желательно выбирать - <a href="https://www.mozilla.org/ru/firefox/windows/" target="_blank">Firefox</a> (он потребляет не много ресурсов ПК и не конфликтует с рабочими базами)<br /><font color="#e53935">!ВАЖНО!: </font><font color="#4db6ac">Выполните синхронизацию в браузере, для этого необходимо авторизоваться в нем по вашей почте, это позволит нам не потерять все настройки после переустановки системы. </font><br />3) Одна из основных утилит которая на необходима - <a href="https://the.earth.li/~sgtatham/putty/latest/w64/putty.exe" target="_blank">Putty</a> ( с его помощью мы работаем на OLT )<br />4) Рекомендуется установить захватчик экрана, например - <a href="https://screenshoter.mail.ru/" target="_blank">ScreenShoter</a> ( он позволит нам делать скриншоты нажатием горячей клавиши и редактировать его как нам захочется )<br />5) Необходимо установить мессенджер - <a href="https://desktop.telegram.org/?setln=ru" target="_blank">Telegram</a> ( это позволит нам быстро реагировать и коммуницировать с рабочими чатами )<br />6) <a href="https://winmtr.ru/" target="_blank">WinMTR</a> - позволит нам выполнять более детальную диагностику к внешним сервисам.<br />

Настройка удаленного доступа на рабочий ПК

Находим утилиту для удаленного подключения и открываем ее

В пункте "Компьютер" вводим IP адрес ПК к которому хотим подключиться
