Настройка часового пояса

После входа в веб-интерфейс открывается мастер быстрой настройки. С его помощью можно задать все необходимые параметры буквально в несколько шагов. Меняем язык страницы с настройками, указываем свой часовой пояс и продолжаем.

Настройка типа подключения

<p>На следующей странице нужно указать параметры для подключения к интернету (интернет-провайдеру). Для начала выбираем тип подключения к интернету, а в следующем окне указываем необходимые данные </p>

После нажатия на галочку, выбираем тип подключения - Динамический

Настройка MAC адреса

МАС адрес устройства оставляем без изменения и нажимаем Далее

Настройка Wi-Fi сети

На странице с настройкой Wi-Fi сети вы можете сменить имя беспроводной сети (SSID) и пароль
